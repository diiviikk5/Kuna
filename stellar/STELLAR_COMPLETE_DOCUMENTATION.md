## How to Use with New Data

### CSV Format Requirements

Your CSV file must have these columns:

```csv
timestamp,x_error,y_error,z_error,clock_error
2024-01-01 00:00:00,0.123456,-0.234567,0.345678,1.234567
2024-01-01 00:15:00,0.124567,-0.235678,0.346789,1.235678
...
```

| Column | Description | Unit | Example |
|--------|-------------|------|---------|
| timestamp | Date and time | ISO 8601 | 2024-01-01 00:15:00 |
| x_error | Radial error | meters | 0.123456 |
| y_error | Along-track error | meters | -0.234567 |
| z_error | Cross-track error | meters | 0.345678 |
| clock_error | Clock bias | nanoseconds | 1.234567 |

### Using Your Own Data

#### Method 1: Scientist Workflow (Recommended)

1. **Go to**: http://localhost:5173 â†’ Click "Scientist Workflow"
2. **Upload**: Drag your CSV file to the upload zone
3. **Validate**: System checks format and shows preview
4. **Configure**: Adjust training parameters if needed
5. **Train**: Click "Start Training"
6. **Evaluate**: View ISRO metrics automatically

#### Method 2: Direct Data Replacement

1. **Prepare** your CSV file in the correct format
2. **Copy** to `public/data/DATA_MEO_Train.csv` (or GEO)
3. **Refresh** the application
4. **Go to** Forecast Lab â†’ Training Window
5. **Select** your orbit type and train

#### Method 3: Programmatic (Advanced)

```javascript
import { loadISROData } from './services/isroDataLoader';
import { trainModel, predictFuture } from './services/aiService';

// Load your data
const data = await loadISROData('MEO', 'Train');

// Train
await trainModel(data, (epoch, loss) => {
    console.log(`Epoch ${epoch}: Loss = ${loss}`);
});

// Predict
const last48Samples = data.slice(-48);
const predictions = await predictFuture(last48Samples);
```

## Quick Start Guide

### Prerequisites

- Node.js 18+ 
- Modern browser (Chrome, Firefox, Edge)

### Installation

```bash
# Clone repository
git clone https://github.com/your-repo/stellar.git

# Install dependencies
cd stellar
npm install

# Start development server
npm run dev

# Open browser
# Navigate to http://localhost:5173
```

### Running Tests

```bash
# Open test page
http://localhost:5173/test-evaluation.html

# This tests:
# - Shapiro-Wilk implementation
# - Residual statistics
# - Q-Q plot generation
```
